INSTALLATION:
------------

1). Install all prerequite modules:
   - jquery_plugin
   - views_ui
   - upload_element
   
2) Copy the rotor directory and all its contents to your modules directory.
   
3) Enable module on the Administer > Site Building > Modules

4) Grant permission to use this module:
   http://example.comadmin/admin/user/permissions#module-rotor

5) Create Rotor items at Content canagement > Create content > Rotor item

6) Place the Rotor block at Site Building -> Blocks

7) Configure the Rotor block at Site Building -> Views

8) Create and configure additional Rotor blocks at Site Building -> Views


To make a view rotate, set its "Style" to "Rotor".
To configure the Rotor view, click the ajax settings wheel next to where it says "plugin style=Rotor"

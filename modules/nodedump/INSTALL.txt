Installation
------------
unarchive the tarball
copy it to your sites/all/modules path
enable the module at /admin/build/modules

Uninstall
---------
An uninstall script is provided, which will remove the configuration variable 
(nodedump_last_nid) used by the module. There are no database tables affected by 
this module.
